StarSpring Auth

A demonstration of session-based authentication and role-based authorization built with the StarSpring framework.

Get Started Sign In
Session Authentication

Secure session-based login using signed cookies powered by itsdangerous. Sessions are tamper-proof.

Role-Based Authorization

Users are assigned USER or ADMIN roles. Admin routes are protected and inaccessible to regular users.

Secure Passwords

Passwords are hashed with SHA-256 before being stored. Plain-text passwords are never saved to the database.
